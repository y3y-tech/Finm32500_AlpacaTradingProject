---
description: Show statistics and breakdown of GitHub issues
---

Display comprehensive statistics about the project's GitHub issues:

**Report includes:**
- Total issues (open/closed)
- Breakdown by priority (P0/P1/P2/P3/P4)
- Breakdown by ownership (AI/Human/Collaborative)
- Breakdown by theme (strategy, testing, risk-management, etc.)
- List of critical P0 issues still open
- List of AI-friendly issues ready to be worked on
- Progress percentage by priority level

**Output format:**
- Summary table with counts
- Visual progress indicators
- Actionable recommendations (e.g., "5 P0 issues remaining")

This command is useful for:
- Daily standup reports
- Sprint planning
- Understanding what work remains
- Identifying bottlenecks

Run this anytime you want a quick overview of project status.
